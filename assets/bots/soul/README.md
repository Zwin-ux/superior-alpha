# Soul Asset Shelf

This folder holds the current Clawd/Hermes identity assets from `clawdbot_codex_soul_brief`.

Use these as the clean-install face of SUPERIOR:

- `clawd-hermes-avatar.png`: shared avatar source.
- `clawdbot-workshop-target.png`: workshop composition target.
- `clawd-hermes-multiplatform-icons.png`: platform icon reference sheet.
- `icons/clawd-windows.ico`: Windows app and MSI icon.
- `icons/chrome-extension-icon-*.png`: Chrome MV3 static icons.

The runtime icon renderer still follows `BotIdentity` for user customization. These files keep every shell from inventing a different default bot.
