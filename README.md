SUPERIOR is a local AI companion and signal-analysis console with a tactile clay UI.

## One-Click Install (Windows)

[**Download Alpha 1 Installer (.exe)**](https://github.com/Zwin-ux/superior-alpha/releases/download/v0.19.0-alpha.1/SUPERIOR-Alpha-1-Installer.exe)

*   **Tactile UI:** Handmade clay workshop built in Godot.
*   **Real-time:** Physical hardware monitors and spinning data spools.
*   **Local:** Privacy-first architecture using a local Node daemon.

## Getting Started

1.  Download and run the `SUPERIOR-Alpha-1-Installer.exe`.
2.  Follow the in-engine "Spore Wake" ritual to set up your bot.
3.  Connect the Chrome extension to link your browser signals.

---

### Local Development
Developer documentation and internal sprint gates are maintained in private local memory. To build from source, ensure you have Node.js >=22 and `corepack` enabled. Run `pnpm install`, `pnpm build`, then `pnpm dev`.
