# Codex Prompt Block

Use this aesthetic block in implementation prompts for SUPERIOR:

```text
Aesthetic direction:
The UI should feel like a claymation desktop game launcher, inspired by old PC game menus like FATE. Avoid SaaS dashboard styling. Use soft clay-like shapes, warm shadows, matte textures, rounded uneven forms, chunky buttons, subtle squash/stretch animation, and a cozy workshop scene. The bot should look like a small handmade clay robot head. Its body/color/eye/skills should update live and appear consistently in the launcher, floating buddy, extension popup, and right-click icon.
```

## Banned Defaults

- Generic SaaS homepage
- Hero headline plus subcopy plus dual CTAs
- Floating dashboard mockup
- Feature-card grid
- Purple AI gradient styling
- Shiny cyberpunk command deck
- Generic vector mascot

## Required Product Phrase

Use `SUPERIOR`, not `Clawdbot Workshop` or `Clawdbot Command Deck`.

Core identity:

```text
A little clay robot you build, then bring into your browser to explain the internet.
```
